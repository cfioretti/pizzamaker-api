# Pizza Maker - API

Laravel api to calculate the right weight of dough in the pan for a pizza

## Requirements 

    Composer
    PHP >= 7.2.5

## Installation

You can install the project via composer:

```sh
git clone git@github.com:cfioretti/pizzamaker-api.git
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```
